# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Reporting A Vulnerability

If you discover a security vulnerability, please do not disclose it publicly.

Contact the ASN Security Team:

[security@asychan.io](mailto:security@asychan.io)

Include:

* Description of vulnerability
* Steps to reproduce
* Potential impact
* Suggested remediation

## Response Timeline

* Acknowledgement within 72 hours
* Initial assessment within 7 days
* Resolution target based on severity

## Responsible Disclosure

We encourage responsible disclosure practices and collaboration with security researchers.

## Scope

This policy covers:

* Smart Contracts
* Website Infrastructure
* APIs
* Governance Systems
* Treasury Management Systems

Thank you for helping keep ASN secure.
